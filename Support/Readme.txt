Support files for SilverMap

+ crossdomain.xml - copy this file to your xServer root to access 
  the xServer directly. For more information see. 
  http://msdn.microsoft.com/de-de/library/cc197955%28VS.95%29.aspx 

+ xmap-empty.properties - an xMapServer profile which disables all
  background content. This profile can be used to create xServer images
  which contain only dynamic content.
